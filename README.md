# 🧮 Ganetrix Calculator

**Ganetrix** - The World's Best Professional Calculator built with Python

## 🌟 Features

✅ **Zero Bugs** - Full error handling and validation  
✅ **Professional Grade** - Production-ready code  
✅ **Beginner Friendly** - Easy to understand and modify  
✅ **7 Operations**:
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Power (**)
- Square Root (√)
- Modulus (%)

✅ **Safety Features**:
- Division by zero protection
- Invalid input handling
- Negative number validation for square root

## 📋 Requirements

- Python 3.6 or higher

## 🚀 How to Use

### Option 1: Run Directly
```bash
python calculator.py
```

### Option 2: Import as Module
```python
from calculator import Calculator

calc = Calculator()
result = calc.add(5, 3)  # Returns 8
result = calc.divide(10, 2)  # Returns 5.0
```
